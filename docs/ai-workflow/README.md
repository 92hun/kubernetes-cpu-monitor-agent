# AI-assisted Development Workflow

이 디렉터리는 OpenAI Codex와 함께 과제를 수행할 때 사용한 단계별 실행 계획과 검증 기록이다.

문서 순서:

1. 환경과 기술 계약
2. Go 핵심 로직 TDD
3. 애플리케이션, 컨테이너, Manifest
4. Minikube 실제 배포와 디버깅
5. 리뷰, 문서화, Public 저장소 제출

각 단계는 목표, 사용한 검증 방식, 완료 조건과 실제 결과를 기록한다. 최종 구현의 요약은 상위 `IMPLEMENTATION_PLAN.md`, 실제 검증 결과는 `VERIFICATION.md`, AI 활용 범위는 `AI_USAGE.md`에서 확인할 수 있다.
